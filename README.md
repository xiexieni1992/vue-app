### 1.简介
> 这是一个基于VUE全家桶的重构项目；

##### 项目用到的技术栈:
> * vue
> * vue-router
> * vuex
> * webpack
> * axios
> * less
> * postman
***

**基础环境**
> node : v8.2.1
npm : 5.3.0

**注:如果项目install有问题,可把对应环境配置成上面相关的环境在尝试**

***启动mock服务器***
```
$ npm install -g json-server
$ npm run server
```

**项目运行**
```
$ npm install
$ npm run dev
$ npm run build
```
