### 1.简介
> 这是一个基于VUE全家桶的重构项目；


##### 项目用到的技术栈:
> * vue
> * vue-router
> * vuex
> * webpack
> * axios
> * less
***

**基础环境**
> node : v8.2.1
npm : 5.3.0

**注:如果项目install有问题,可把对应环境配置成上面相关的环境在尝试**

**项目运行**
```
$ npm install
$ npm run dev
$ npm run build
```



### .命名规范
> 组件样式命名: 统一采用 ub 前缀 如: ub-header
> 页面样式命名: 统一采用 page 前缀 如: page-login
> 文件命名: 统一采用-命名方式: 如: user-help
> 样式命名: 统一采用-命名方式: 如: .user-help
> 图片命名: 统一采用-命名方式: 如: .icon-help.png
